### 分析

题目大意就是一个容量限制为 **m** 的栈，一组数 1,2,3,…,n 顺序入栈，随机出栈，问出栈序列是、否可能? 

- 找规律：一时半会找不出来，放弃； 
- **程序模拟**方法：
	- 使用 vector int 数组存储输入序列，自定义栈 s，让 1,2,…n 顺序入栈，定义 idx 指示当前 vector 数组下标， 当 vector[idx] 等于栈顶元素（s.top ()）且栈 s 不为空，不断出栈且 idx+1 直到不等为止; 当 1…n 入栈结束，或者栈 s 已满，退出入栈循环，最后判断 idx 与 n 的大小关系决定该顺序是否可能？
	- exit(-1) 的部分，可能使用 try catch 语句重写；
